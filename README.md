# Employee-Project-Client Management System (SQL Project)

## Description
This project is an Employee & Project Management System using **8 tables**:
DEPARTMENT, EMPLOYEE, CLIENT, PROJECT, TECH_STACK, PROJECT_TECH, SALARY, PERFORMANCE.

It demonstrates advanced SQL concepts including:
- Joins (INNER, LEFT)
- Subqueries and nested queries
- Aggregate functions
- Many-to-many relationships (PROJECT_TECH)

## Features
- Retrieve employee details along with department info
- Track projects, clients, and team leads
- Map technologies to projects
- Employee salary and payroll details
- Performance review and promotion tracking
- Advanced queries: second highest salary, employees above certain pay

## Setup Instructions
1. Install Oracle Database or any SQL IDE.
2. Clone the repository:
   ```bash
   git clone https://github.com/shangarwarsandesh/employee-project-client-management-system.git
   ```
3. Run `create_tables.sql` to create all tables.
4. Run `insert_data.sql` to populate sample data.
5. Execute `queries.sql` to see example queries in action.
